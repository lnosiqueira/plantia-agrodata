# PlantIA Agrodata

Projeto para monitoramento agrícola com dashboard interativo, gráficos avançados e integração com previsão do tempo.

## Estrutura
- src/: Código principal
- data/: Banco de dados
- assets/: Imagens e estilos
- tests/: Scripts de teste

## Como executar
```bash
pip install -r requirements.txt
python src/main.py
```
